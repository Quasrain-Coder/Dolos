## 1. 游戏简介

**Dolos（瞎掰王）** 是一款多人实时派对游戏。核心玩法为**"谁是老实人"**：隐藏角色推理——老实人说真话，大聪明来破案，瞎掰人编假话忽悠。

游戏没有固定回合上限，想玩多久玩多久，由房主决定何时结束。

**最低人数：** 2 人（推荐 4-8 人）  
**设备支持：** 手机 / 平板 / PC 浏览器  
**每局时长：** 自由控制，通常 15-40 分钟

## 2. 准备开始

### 2.1 创建房间（房主）

1. 打开游戏页面
2. **登录**（可选）或直接输入**昵称**
3. 点击 **「✨ 创建新房间」**
4. 系统生成 **4 位房间码**（如 `AB3K`）
5. 将房间码分享给其他朋友

### 2.2 加入房间（玩家）

1. 打开游戏页面
2. 输入房主分享的 **4 位房间码**
3. 点击 **「加入」** 进入房间大厅
4. 在大厅可以看到玩家列表

### 2.3 开始游戏

- 所有玩家进入房间后，在大厅可以看到玩家列表
- 当至少 **2 人**进入房间后，房主点击 **「🎮 开始游戏」**
- 游戏随即进入第一个回合

## 3. 角色说明

系统每回合从所有玩家中**随机分配**三个隐藏角色：

| 角色 | 人数 | 特殊能力 | 限制 |
|------|------|----------|------|
| 🤝 **老实人** | 1 人 | 知道真定义 | 必须提交真定义，不可编假话 |
| 🕵️ **大聪明** | 1 人 | 选题 + 投票 + 猜人 | 不知道真定义，需要推理 |
| 🎭 **瞎掰人** | 剩余 | 无 | 不知道真定义，编假答案忽悠 |

> 🔒 角色是**保密的**——只有你自己知道自己的角色。

## 4. 回合流程

流程全自动推进：

**阶段一：大聪明选题 `SELECTING`**

- 系统随机分配角色
- 大聪明看到 **3 道备选题**（含分类标签）
- 可以 **🔄 换一批** 刷新备选题
- 点击题卡选择，再点 **✅ 就选这个** 确认
- 其他玩家等待大聪明选题

**阶段二：编答案 `ANSWERING`**

- 所有玩家看到题目
- **老实人**的输入框自动预填真定义，确认提交即可
- **瞎掰人**和大聪明编假答案（大聪明也不知道真答案！）
- 目标：编得**足够逼真**，让大聪明误以为是真答案
- 全员提交后自动进入投票

**阶段三：投票 `VOTING`**

- 所有答案匿名混排
- 大聪明从答案中选出**自己认为是真定义的一项**
- 选错可以重试，选对进入揭晓

**阶段四：揭晓 `REVEALING`**

- 显示正确答案、所有答案作者
- 大聪明猜对 +2 分，被骗选错的答案作者各 +1 分
- 显示本回合得分明细和排行榜

**阶段五：回合结束 `READY`**

- 所有玩家点击 **✅ 就绪** 进入下一轮
- 角色**重新随机分配**
- 房主可随时点击 **🏁 结束游戏**

## 5. 计分规则

得分实时累计，排行榜在页面顶部随时可见。

| 情况 | 得分 |
|------|------|
| 🎯 大聪明投中真答案 | +2 |
| 🎣 骗到大聪明投票 | +1 / 人 |
| 🕵️ 大聪明猜中老实人 | +3 |

---

## 6. 游戏结束

- 游戏无固定结束条件，**房主**在任意回合揭晓后点击 **「🏁 结束游戏」**
- 结束后展示最终排行榜：🥇🥈🥉
- 点击「返回首页」即可开启新的一局

## 7. 操作指南

### 界面布局

```
┌──────────────────────────┐
│  🤝 你的角色：老实人       │  ← 角色标签（仅自己可见）
├──────────────────────────┤
│  🏅 排行榜                │
├──────────────────────────┤
│  圆桌 — 玩家状态卡片       │  ← 提交后变绿 ✅，投票后 🗳️
├──────────────────────────┤
│  主要游戏区域              │
│  · 大聪明选题 / 编答案     │
│  · 投票                  │
│  · 揭晓                  │
├──────────────────────────┤
│  就绪 / 结束游戏           │
└──────────────────────────┘
```

### 大聪明选题面板

- 显示 3 道备选题（含分类标签如"冷知识""科技术语"）
- 🟡 金色边框 = 选中
- 🔄 换一批 = 刷新题目
- ✅ 就选这个 = 确认
- 其他玩家看到"🕵️ 大聪明正在选题..."

### 断线重连

- 断开连接后 **2 秒自动重连**
- 重连后使用相同身份回到游戏，角色信息也会恢复
- 断线期间该玩家标记为「离线」状态

## 8. 边界情况

| 场景 | 系统处理 |
|------|----------|
| 断线 | 自动标记离线，同 ID + Token 可重连恢复 |
| 老实人/大聪明断线 | 回合继续，其他玩家体验不受影响 |
| 投票未投 | 系统自动跳过 |
| 房间人走光 | 房间自动清理 |
| 人数不足 | 在线玩家少于 2 人，游戏暂停 |

## 9. FAQ

### Q: 最少几个人可以玩？
最少 2 人。推荐 4 人以上，人越多推理越有趣。

### Q: 老实人可以编假话吗？
不可以。老实人的输入框会被真定义预填且不可编辑。

### Q: 大聪明怎么知道谁是老实人？
大聪明不知道！他和其他人一样只能看到题目，需要根据各选项的内容来推理判断。

### Q: 大聪明猜对老实人有什么好处？
猜对 +3 分！

### Q: 可以自己出题吗？
当前使用内置题库（冷知识、历史、科技术语、网络梗等 454 题）。未来会支持玩家贡献新题。

### Q: 为什么没有经典模式了？
经典模式已被移除，专注于"谁是老实人"这一更有趣的社交推理玩法。
